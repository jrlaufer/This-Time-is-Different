# This-Time-is-Different
Data and code Repository for This Time is Different: March for Our Lives and 2018 Midterm Election paper (Nature submission)

The base dataset is the Congressional Election Survey (CES) Cumulative Common Content 2006-2024 module available on Dataverse: https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/II2DB6

• Cleaning code for variable transformations: > Nature cleaning code.R <br>
• Analysis code for DiD and synthetic controls: Replication code: > FINAL MFOL.R<br>
• Final dataset that was used in DiD analysis: > finaldata.zip<br>
• March for Our Lives county list (FIPS): > MFOL Visit List by County.csv<br>
• GIS file with 100 mile radius and control county mapping: > Controlcounties100miles.dbf<br>
